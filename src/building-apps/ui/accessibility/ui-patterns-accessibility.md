# UI patterns accessibility

Most OutSystems widgets and OutSystems UI patterns meet WCAG 2.2 AA by default. However, a few patterns rely on complex interactions that assistive technologies don't always interpret correctly, so they need additional configuration to stay accessible.

## Known issues

The UI Patterns listed in this section require additional work to be fully compliant with WCAG 2.2 AA. All other patterns are WCAG 2.2 AA compliant by default.
