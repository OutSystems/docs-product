# Range Slider Interval Reference

## Events

**Event Name** |  **Description** |  **Mandatory**  
---|---|--- 
|OnChange  |  Action to execute after selecting a new value on the slider. Returns the new IntervalStart and the new IntervalEnd.  |  True|
