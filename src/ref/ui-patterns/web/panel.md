## Advanced use case

### Add box-shadow on Panel

1. Set the ExtendedClass parameter to `shadow-l`.

    ![Screenshot showing a Panel with the box-shadow effect applied in a Traditional Web App](images/panel-3-ss.png "Panel with Box-Shadow")

1. Publish and test.
